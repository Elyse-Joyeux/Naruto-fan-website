# Naruto Fan Website

A visually rich Naruto fan website built with React, Vite, and Tailwind CSS. The project showcases legendary shinobi, hokage archives, major wars, missions, videos, and lore in a polished, interactive presentation.

## Features

- Responsive Naruto-themed landing page with animated sections
- Hero search and filtering by village and rank
- Interactive hero, war, and mission cards
- Embedded YouTube video gallery with watched status
- Smooth transitions, hover effects, and modern UI styling
- Background music toggle and ambient particle animation
- Accessible navigation and keyboard-friendly focus states

## Tech Stack

- React
- Vite
- TypeScript
- Tailwind CSS
- Radix UI components
- Lucide icons

## Project Setup

1. Install dependencies:

   ```bash
   npm install
   ```

2. Run local development server:

   ```bash
   npm run dev
   ```

3. Build for production:
   ```bash
   npm run build
   ```

## Folder Structure

- `src/app` - main application and UI components
- `src/data` - Naruto content and metadata
- `src/styles` - global styling and theme configuration
- `public` - static assets like images and audio files
